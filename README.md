# FrustrationStick

### ゲームプレイはこちら
・[イライラ棒風ゲーム](https://unityroom.com/games/frustrationstick)

### 操作方法
・ボタン：左クリック
・キューブ移動：マウス操作
・赤い四角：ゴール
・黒いもの：キューブが触れたらやり直し
